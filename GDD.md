\# Arcade Node++ Game Design Document

\## Elevator Pitch \& Core Pillars

\### Elevator Pitch

Arkanoid-like мобильная игра для Android с доработанным игровым процессом (пока что), а также с roguelite режимом игры (в будущем). В roguelite режиме можно будет выбрать набор уровней, после прохождения каждого из которых (уровней) будут даваться на выбор 1 из 3 усилений. В конце каждого из наборов уровней будет Босс. 

\### Core Pillars

1\) Понятный геймдизайн - интерфейс будет устроен таким образом, чтобы игрок на интуитивном уровне понимал игровой процесс.

2\) Улучшенный баланс в сравнении с остальными играми в жанре. 

3\) Стильный визуальный стиль.

4\) Совершенная техническая часть. 

\## Core Gameplay Loop

С помощью слайдера внизу экрана игрок двигает ракетку, которой нужно отбивать шарик. Шарик может попадать по блокам на уровне, таким образом уничтожая их. Чтобы пройти уровень, нужно уничтожить все уничтожаемые блоки на уровне. С блоков могут выпадать усиления или ослабления. Если игроку не удается отбить шарик, он теряет 1 жизнь. Проигрышь настаёт тогда, когда игрок теряет все жизни. 

\## Technical Architecture Constraints

1\) Платформа - Android (Touchscreen)

2\) Разрешение - Portrait 720 X 1280 (Масштабируется)

3\) Отрисовщие - Compatibility (OpenGL)

4\) Архитектура - Event Bus для связи UI и игры, Object Pooling для снарядов и частиц. State Machine для логики Боссов.
